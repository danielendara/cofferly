# Code of Conduct

Atlas Wallet should be a calm, useful, family-friendly project.

## Expected Behavior

- Be respectful and constructive.
- Keep discussion focused on improving the app.
- Remember that this project handles family and child-related workflows, so privacy and care matter.

## Unacceptable Behavior

- Harassment, insults, or hostile comments.
- Posting private information.
- Pressuring maintainers to accept changes.
- Adding or requesting features that compromise child privacy.

## Enforcement

Maintainers may close issues, reject pull requests, remove comments, or block participants when needed to keep the project healthy.
